## Installation

Install the dependencies and devDependencies and start the server.

```sh
git clone https://github.com/yasirarafat28/food-delivery-nodejs
cd food-delivery-nodejs
npm i
npm run build
md-seed run
npm run start:prod
```

For production environments...

```sh
NODE_ENV=prod
```
